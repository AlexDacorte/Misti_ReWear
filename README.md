# 🏔️ Misti_ReWear

**Misti_ReWear** is a web application built with **React** focused on the **sale of second-hand clothing**, combining **sustainable fashion** with a **neo-brutalist design aesthetic** inspired by the city of **Arequipa, Peru** and the Misti volcano.

The project promotes conscious consumption by giving clothes a second life while offering an intentionally raw, bold, and functional user experience.

---

## 🧠 Concept

> Rewear. Reuse. Resist fast fashion.

Misti_ReWear is built around three core ideas:

- Circular fashion
- Local identity
- Radical simplicity through neo-brutalism

Every garment tells a story. Every interface choice is deliberate.
